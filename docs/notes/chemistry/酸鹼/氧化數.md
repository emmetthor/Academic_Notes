# 氧化數

## 介紹

氧化數是判斷氧化還原反應的工具，其口決是：`0 電 1 2 3 氟 氫 氧`。

氧化數實際上是一種假設，若所有電子都跑到電負度高的原子去，那每個原子的電荷會是多少。

氧化數可以幫助判斷氧化還原反應式中，究竟是哪些原子被氧化，哪些被還原。

### 含意

- `0`：元素不帶電，因此氧化數為 `0`。
- `電`：帶電物質的氧化數就是其電量。
- `1`：代表 1A 族，也就是，1A 族的氧化數是 `+1`。
- `2`：代表 2A 族，也就是，2A 族的氧化數是 `+2`。
- `3`：代表 3A 族，也就是，3A 族的氧化數是 `+3`。
- `氟`：F 的氧化數是 `-1`。
- `氫`：H 的氧化數是 `+1`。
- `氧`：O 的氧化數是 `-2`。
