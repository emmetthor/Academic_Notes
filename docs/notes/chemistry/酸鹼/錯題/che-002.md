# che-002

## 題目

下列 5 種常見反應或過程中，哪些涉及氧化還原反應？（應選 2 項）【110.試辦考試】

(A) 光合作用
$\ce{6CO2(g) + 6H2O(l) -> C6H12O6(s) + 6O2(g)}$

(B) 汽車觸媒轉化作用
$\ce{2NO2(g) -> N2(g) + 2O2(g)}$

(C) 暫時硬水煮沸
$\ce{Ca(HCO3)2(aq) -> CaCO3(s) + CO2(g) + H2O(l)}$

(D) 鐘乳石的形成
$\ce{Ca(OH)2(aq) + CO2(aq) -> CaCO3(s) + H2O(l)}$

(E) 小蘇打（碳酸氫鈉）受熱分解
$\ce{2NaHCO3(s) -> Na2CO3(s) + CO2(g) + H2O(l)}$

## 解法

::: details

(A) $\ce{CO2}$ 中的碳從 $+4$ 變成 $+0$（在 $\ce{C6H12O6}$）中。有氧化還原。

(B) $\ce{NO2}$ 中的氧從 $-2$ 變成 $+0$（在 $\ce{O2}$ 中）。有氧化還原。

(C) 完全沒有電荷轉移。

(D) 完全沒有電荷轉移。

(E) 完全沒有電荷轉移。

:::
