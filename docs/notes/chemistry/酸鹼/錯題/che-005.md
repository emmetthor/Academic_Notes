# che-005

## 題目

把一根鐵絲浸泡在鹽酸水溶液中其表面產生氣泡，則下列相關敘述何者錯誤？

(A) 反應產生的氣體為氫氣

(B) 還原劑為 $\ce{Fe}$

(C) 氧化劑為 $\ce{Cl−}$

(D) 反應後鐵絲質量減輕

(E) 反應後溶液中的 $\ce{[Fe^{2+}] + [H+] = [Cl−] + [OH−]}$

## 解法

::: details

先列出化學式：$\ce{Fe + 2HCl -> FeCl2 + H2}$

與淨離子反應式：$\ce{Fe + 2H+ -> Fe^{2+} + H2}$

(A) 應是產生氫氣。

(B) 氫的氧化數從 $+1$ 變成 $0$，被鐵還原，正確。

(C) 根據淨離子反應式，氯只是旁觀離子。

(D) 鐵被轉成鹽類解離在水裡，確實使鐵的質量減少。

(E) 若是電荷守恆，則 $\ce{Fe^{2+}}$ 可以提供兩個 +，因此電荷守恆應寫成： $\ce{2[Fe^{2+}] + [H+] = [Cl−] + [OH−]}$

:::
