# 含氮鹼基

::: tip 重點整理

- 在 RNA 中， T 被替換成 U。
- AT 以**兩氫鍵**連接； CG 以**三氫鍵**連接。

:::

## 定義

- DNA、RNA 中決定基因的密碼。
- 影響 DNA 轉錄，RNA 轉譯與蛋白質密碼子。

## 類型

### 嘌呤 (Purines)

- 腺嘌呤 (Adenine, A)![腺嘌呤](https://upload.wikimedia.org/wikipedia/commons/thumb/d/db/Adenine.svg/1280px-Adenine.svg.png)
- 鳥糞嘌呤 (Guanine, G)![鳥糞嘌呤](https://upload.wikimedia.org/wikipedia/commons/thumb/3/3f/Guanin.svg/1920px-Guanin.svg.png)

### 嘧啶 (Pyrimidines)

- 胸腺嘧啶 (Thymine, T)![胸腺嘧啶](https://upload.wikimedia.org/wikipedia/commons/e/ef/Thymine_chemical_structure.png)
- 胞嘧啶 (Cytosine, C)![胞嘧啶](https://upload.wikimedia.org/wikipedia/commons/thumb/1/10/Cytosine_chemical_structure.png/960px-Cytosine_chemical_structure.png)
- 尿嘧啶 (Uracil, U)![尿嘧啶](https://upload.wikimedia.org/wikipedia/commons/thumb/b/b4/Uracil.svg/960px-Uracil.svg.png)

::: info 為何嘌呤是嘌呤，嘧啶是嘧啶？

嘌呤有兩環，而嘧啶只有一環。

:::
