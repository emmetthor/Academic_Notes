# 遺傳學史

## 創造論

### 主旨

生命過於完美，必定有一個造物者創造所有生物。

### 觀點

- **生物不會改變**，畢竟是由造物者創造的。
- 所有物種變化都是**經過重新設計而創造出**的，絕非原有物種自行改變。

### 信奉者

- 亞里斯多德
- 林奈：使用**階梯級位**與**二名法**分類生物。
- 居維業：提出**災變說**，認為物種器官**牽一髮而動全身，物種變異不可能發生**。

::: info 階梯級位

也就是 `界門綱目科屬種`，這是由林奈提出的喔。

:::

::: info 二名法

二名法由 `屬名 + 種小名` 組成，例如 `Homo Sapiens` 就是現代智人的學名。

:::

::: info 災變說

由於物種不可能發生變異，而沉積岩內又有許多史前時代的化石，因此認為：**必定有多次大災難使物種滅絕**，造物者又創造出新的物種，才造成化石生物與地表生物不同。

:::

## 演化理論啟蒙

布豐：研究化石與解剖學，提出**生物會隨時間演變**。

## 用進廢退與獲得性遺傳

### 主旨

生物**為了適應環境，而造成性狀變化**。

### 觀點

- 生物經過**漫長時間累積變異**，造成物種演化。
- **用進廢退論**：生物**性狀改變是為了適應環境**，也就是，環境改變是性狀改變的起因。
- 獲得性遺傳：親代後天的性狀改變可以遺傳給子代。

::: info 用進廢退例子：長頸鹿的長脖子

由於環境中，較矮的樹葉都被吃光了，矮頸鹿想要吃到高處的葉子，於是使勁伸長脖子，便使脖子變長了。

符合：`環境改變 -> 性狀改變`。

:::

### 提出者

拉馬克。

### 質疑

魏斯曼老鼠檢尾實驗，對獲得性遺傳提出質疑。

::: info 老鼠剪尾實驗

模擬環境改變，將每一代老鼠的尾巴剪去，觀察後代的尾巴是否有縮短。而實驗結果為非。

當然，這只能證明人為剪除的性狀改變無法遺傳，而不代表獲得姓遺傳完全錯誤。

:::

## 天擇說

### 主旨

**環境會篩選生物，簡稱「物競天擇，適者生存」**。

### 觀點

- **漸變與累世修飾**：物種緩慢累積變異是為「漸變」、「累世修飾」。
- **物種同源**：認為**所有生物皆有共同祖先**。
- **天擇說**：生物個體間有變異，經過環境選擇後，留下適合生存的性狀。

::: info 天擇說例子：長頸鹿

自然界中本就有長頸鹿與矮頸鹿，由於矮處的葉子都被吃光，只留下高處的葉子，因此**矮頸鹿無法獲得食物繁衍，只剩長頸鹿繼續存活**。

符合：`物種多樣 -> 環境篩選 -> 適者生存`。

:::

### 提出者

達爾文，華萊士。

### 參考文獻

- 馬爾薩斯《人口論》：資源不足時，生物會發生競爭，並非所有生物都會活下來。
- 萊爾《地質學原理》：現在是通往過去的鑰匙。

## 現代綜合演化理論

### 解釋獲得性遺傳

若變異發生於體細胞時，性狀改變不會遺傳給子代；若**發生於生殖細胞，便會遺傳給子代**。

### 解釋天擇說

不同個體擁有不同等位基因，**若某個體比較強勢，便更有機會使基因流傳給子代，這便是天擇**。
