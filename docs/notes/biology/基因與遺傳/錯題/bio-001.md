# bio-001

## 題目

甲股：`ACCGATGCCTACG`；乙股：`TGGCTACGGATGC`

以下描述何者正確？

(A) 甲和乙為兩個等位基因

(B) 若此為一基因的部分，甲、乙均可為模版進行轉錄作用

(C) 若其一姊妹染色分體具有此序列，則另一姊妹染色分體也具此序列

(D) 若一同源染色體具有此序列，則另一同源染色體也具有此序列

## 解法

::: details

(A) 甲乙是 DNA 的兩股。

(B) 轉錄時，**只有其中一條會被當成模板股**，且每次轉錄皆會穩定選擇此條。

(C) 姊妹染色分體互為複製 DNA，因此必定具有相同的鹼基序列。 :white_check_mark:

(D) 同源染色體不一定具有相同的基因，因此鹼基序列也不一定相同。

:::
