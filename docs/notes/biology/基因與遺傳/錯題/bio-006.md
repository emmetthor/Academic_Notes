# bio-006

## 題目

下列關於各種性狀的說明，哪些正確？

(A) 膚色的表現為多基因遺傳，不受環境影響

(B) 植物的花色可能受完全顯性遺傳、不完全顯性遺傳，或土壤性質的影響

(C) 喜馬拉雅兔的毛色為基因突變的結果

(D) 人類鐮刀型血球貧血症為基因突變的結果

(E) 若媽媽為血友病患者，兒子也會是

## 解法

::: details

(A) 人會曬黑，這是由於黑色素堆積形成的。與環境有關。

(B) 正確。

(C) 在考試時，通常喜馬拉雅兔的毛色是由溫度影響的。

(D) 正確。

(E) 正確。血友病是 X 染色體的隱性遺傳，因此媽媽的 X 染色體全部都是隱性，兒子也必定受隱性 X 影響。

:::
