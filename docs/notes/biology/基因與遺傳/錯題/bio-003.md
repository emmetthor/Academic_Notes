# bio-003

## 題目

魏斯曼將老鼠尾巴剪除後，讓此老鼠交配產生後代，生出的後代仍有尾巴，連續處理到第二十一代，發現每一代的老鼠尾巴都沒有變短，關於此實驗的描述，何者正確？

(A) 此實驗可以檢測獲得性遺傳是否成立

(B) 此實驗可以說明老鼠尾巴是有用的構造

(C) 此實驗會改變老鼠生殖細胞的 DNA

(D) 此實驗推翻拉馬克的理論，較符合神創論物種特徵不變的道理

## 解法

::: details

(A) 正確。

(B) 人工把尾巴減去，目的就是讓尾巴成為無用的構造。

(C) 尾巴不是生殖細胞。

(D) 只能說明獲得性遺傳在 21 代之前都沒有顯現，並不能完全否定拉馬克的學說。

:::
