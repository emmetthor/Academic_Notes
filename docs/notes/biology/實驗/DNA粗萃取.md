# DNA 粗萃取

想要大致取出 DNA，必須移除細胞壁、細胞膜、核膜與附著之蛋白質，因此實驗流程如下：

## 流程

### 1. 去除細胞壁

若生物具有細胞壁，最快速的做法就是**打碎**。

### 2. 去除細胞膜與核膜

細胞膜與核膜由雙層磷脂質組成，也就是油脂。因此可以**用介面活性劑去除細胞膜與核膜**。

### 3. 去除附著之蛋白質

由於染色質上的 DNA 與蛋白質附著很緊密，因此最好的做法是**將 DNA 溶解出來**。

因此**加入食鹽水**，讓其中的 $\ce{Na+}$ 吸引帶負電的 DNA；$\ce{Cl-}$ 吸引帶正電的蛋白質。

::: info 為何 DNA 與蛋白質有電荷？

DNA 上有磷酸根，而磷酸根帶負電，故 DNA 帶有負電荷。

而整個染色質又必須電中性，於是必須由蛋白質帶正電來中和電性，故蛋白質帶正電荷。

:::

當 DNA 被溶解後，就可以去除蛋白質，於是**加入蛋白酶**。

### 4. 析出 DNA

由於 DNA 帶負電，他會與水（極性分子）吸在一起，讓溶解度變高。

因此**加入 95% 酒精**，可以使溶液極性降低，降低 DNA 溶解度，析出於酒精之中。
