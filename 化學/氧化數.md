# 氧化數

## 介紹

`0, 電, 1A, 2A, 3A, F, H, O`

是從「最明確」排到「比較常見但有例外」。

| 口訣 | 意涵 | 例子 |
| --- | --- | --- |
| 0 | 元素氧化數必定是 0 | $\ce{Ag, Cl2}$ |
| 電 | 離子氧化數就是電荷數 | $\ce{Fe^{2+}, Fe^{3+}}$ |
| 1A | 1A 族金屬通常為 `+1` | $\ce{NaCl, NaOH}$ |
| 2A | 2A 族金屬通常為 `+2` | $\ce{MgCl2, MgO}$ |
| 3A | 2A 族金屬通常為 `+3` | $\ce{Ag(OH)3}$ |
| F | F 通常是 `-1` | $\ce{OF2}$ |
| H | H 常常是 `+1` | $\ce{H2O}$ |
| O | O 常常是 `-2` | $\ce{H2O}$ |
