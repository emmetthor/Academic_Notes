# 平衡化學式

## 觀察 + 係數法

Too easy.

## 消除旁觀離子

對於一些複雜的反應式，有一些離子其實只是為了配成鹽類而存在，並無實質參與反應，因此這些離子便可以先削掉。

---

### 例子

#### 刪除 NO3-

$\ce{Zn + HNO3 -> Zn(NO3)2 + NH4NO3 + H2O}$

先把可拆的離子拆開：

$\ce{Zn + H+ + NO3- -> Zn^{2+} + 2NO3- + NH4+ + NO3- + H2O}$

會發現只有一個 $\ce{NO3-}$ 會被拆開變成 $\ce{NH4+}$，其他都在湊數，因此把他們刪了：

$\ce{Zn + H+ + NO3- -> Zn^{2+}  + NH4+ + H2O}$

平衡後：

$\ce{4Zn + 10H+ + NO3- -> 4Zn^{2+} + NH4+ + 3H2O}$

最後放入被刪除的離子：

$\ce{4Zn + 10HNO3- -> 4Zn(NO3)2 + NH4NO3 + 3H2O}$

請注意，簡式的 $\ce{NO3-}$ 會消失，因為他已經參與了 $\ce{NO3- -> NH4+}$，而且已經合併在 $\ce{10HNO3}$ 裡面了。

---

#### 刪除 K2 SO4 2-

$\ce{K2Cr2O7 + H2SO4 + H2S -> KHSO4 + Cr2(SO4)3 + H2O + S}$

先把可拆的離子拆開：

$\ce{2K+ + Cr2O7^{2-} + 2H+ + SO4^{2-} + 2H+ + S^{2-} -> K+ + H+ + SO4^{2-} + 2Cr^{3+} + 3SO4^{2-} + H2O + S}$

把能刪的刪了：

$\ce{Cr2O7^{2-} + H+ + S^{2-} -> Cr^{3+} + H2O + S}$

> 為何 H+ 留下來了？因為他的氫是唯一留在反應物的。

平衡：

$\ce{1Cr2O7^{2-} + 14H+ + 3S^{2-} -> 2Cr^{3+} + 7H2O + 3S}$

最後放入被刪除的離子：

$\ce{1K2Cr2O7 + 5H2SO4 + 3H2S -> 2KHSO4 + Cr2(SO4)3 + 7H2O + 3S}$
